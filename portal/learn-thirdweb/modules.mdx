## Create a Module in thirdweb project

With thirdweb you can mint NFT's, create drops and build your own marketplace. All of this is done inside Projects and Modules. In this guide we'll deploy a module on your thirdweb project. If you don't know what **Projects** and **Modules** are, then [click here.](/learn-thirdweb/projects-and-modules)

### Navigate to Project

![](/assets/portal/learn-thirdweb/projects-and-modules/My_project_2.png)

### Select Module

Pick any module you want. In this tutorial we want to create a drop, so we need the 'Drop' module for this.

![](/assets/portal/learn-thirdweb/projects-and-modules/Drop.png)

### Customize Module

Go ahead and fill in a name for this module and optionally you can add a 'Royalty'.

![](/assets/portal/learn-thirdweb/projects-and-modules/add-module.png)

### Confirm Transactions

You will be asked to confirm two transactions again to create the module.

![](/assets/portal/learn-thirdweb/projects-and-modules/adding-module.png)

![](/assets/portal/learn-thirdweb/projects-and-modules/metamask4.png)

![](/assets/portal/learn-thirdweb/projects-and-modules/metamask5.png)

</ImageSideToSide>

### Module Deployed

The module is created and deployed on to the blockchain.
